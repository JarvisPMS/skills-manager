# skills-manager

用于管理 Agent Skills 的三件套：创建、安装、列出，并同时支持 AgentSkills / Claude / Codex 三种规范目录。

## 包含内容

- skill-creator：手把手引导你创建技能
- skill-installer：帮你安装已经存在技能
- skill-lister：扫描并列出已安装技能

## 快速使用

根据这个文档进行安装。或者把这个文档扔给 Claude Code 让它帮你完成。
