# ASCII 艺术字母转换器

这个技能可以将普通的字母或单词转换为 ASCII 艺术字形式，让文本更加醒目和富有装饰性。

## 使用场景

- 生成醒目的标题或横幅文本
- 为文档或代码注释添加装饰性标题
- 创建视觉吸引力强的终端输出
- 制作艺术化的欢迎信息或标语
- 为 README 文件添加美观的项目名称

## 功能说明

### 核心功能

1. **字母转换**: 将单个字母或完整单词转换为 ASCII 艺术字
2. **多种风格**: 支持多种 ASCII 艺术字体风格（标准、粗体、3D 等）
3. **智能排版**: 自动处理字符间距和对齐
4. **长度适配**: 根据输入长度选择合适的字体大小

### 支持的字体风格

- **标准风格** (Standard): 经典的 ASCII 艺术字，适合大多数场景
- **粗体风格** (Bold): 更加醒目的加粗效果
- **3D 风格** (3D): 立体感的艺术字
- **简约风格** (Simple): 简洁的装饰线条
- **花体风格** (Fancy): 装饰性强的艺术字

## 使用方法

当用户说出以下内容时，自动启用此技能：
- "把 [文字] 转换成 ASCII 艺术字"
- "生成 [文字] 的 ASCII 艺术"
- "给我一个 [文字] 的横幅"
- "用 ASCII 艺术显示 [文字]"
- 或任何类似的请求

## 转换步骤

1. **接收输入**: 获取用户想要转换的字母或单词
2. **验证输入**:
   - 检查输入长度（建议不超过 20 个字符以保证显示效果）
   - 仅支持英文字母、数字和常见符号
   - 空格会被正确处理
3. **选择风格**:
   - 如果用户指定了风格，使用指定的风格
   - 否则使用标准风格
4. **生成艺术字**: 使用 ASCII 字符构建艺术字形式
5. **输出结果**: 以代码块形式输出，保持格式完整

## 技术实现指导

### ASCII 艺术字生成方法

为 Claude 提供以下实现指导：

1. **字符映射表**: 为每个字母和数字创建对应的 ASCII 艺术字模式
2. **行拼接**: 将多个字符的每一行水平拼接形成完整单词
3. **格式保持**: 使用代码块（```）输出以保持格式
4. **边界处理**: 适当添加空行和边距

### 常用字符高度

- 标准风格: 5-7 行高
- 简约风格: 4-5 行高
- 3D 风格: 5-6 行高
- 粗体风格: 6-8 行高

## 注意事项

1. **长度限制**:
   - 建议单词长度不超过 20 个字符
   - 过长的文本可能导致显示效果不佳
   - 如果输入过长，建议拆分为多行

2. **字符支持**:
   - 主要支持英文大写字母 (A-Z)
   - 支持数字 (0-9)
   - 支持常见符号 (! ? . , - _)
   - 小写字母会自动转换为大写

3. **显示环境**:
   - 使用等宽字体（monospace）查看效果最佳
   - 确保终端或编辑器宽度足够
   - 避免自动换行影响显示

4. **风格选择**:
   - 如果用户未指定风格，默认使用标准风格
   - 根据使用场景推荐合适的风格
   - 可以为用户展示多种风格供选择

## 扩展功能

可以根据需要添加以下功能：

- **颜色支持**: 使用 ANSI 颜色代码添加色彩
- **边框装饰**: 为艺术字添加装饰性边框
- **居中对齐**: 在指定宽度内居中显示
- **导出文件**: 将结果保存为文本文件
- **自定义字体**: 支持用户上传自定义字体模板
