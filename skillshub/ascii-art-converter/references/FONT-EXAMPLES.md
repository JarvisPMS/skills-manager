## 使用建议

### 1. 字符对齐
- 确保每个字符的高度一致
- 使用等宽字体显示
- 保持字符间距统一（通常 1-2 个空格）

### 2. 行高标准
- 标准风格: 5 行
- 简约风格: 4 行
- 3D 风格: 5 行
- 粗体风格: 6 行
- 花体风格: 3 行

### 3. 拼接规则
当生成多字符单词时：
1. 分别获取每个字符的所有行
2. 按行号水平拼接
3. 字符之间保持适当间距
4. 确保顶部和底部对齐

### 4. 特殊处理
- 空格: 用固定宽度的空白填充
- 小写转大写: 自动转换
- 不支持的字符: 用问号或空格替代

## 实现参考

```python
# 伪代码示例
def generate_ascii_art(text, style='standard'):
    # 1. 转换为大写
    text = text.upper()

    # 2. 获取字体映射
    font_map = get_font_map(style)

    # 3. 逐行拼接
    lines = []
    for line_num in range(font_height):
        line = ""
        for char in text:
            if char in font_map:
                line += font_map[char][line_num] + "  "
            else:
                line += " " * char_width + "  "
        lines.append(line)

    # 4. 返回结果
    return "\n".join(lines)
```

## 常见问题

**Q: 为什么有些字符显示不正确？**
A: 确保使用等宽字体（如 Consolas, Courier New, Monaco）

**Q: 如何调整字符间距？**
A: 修改拼接时添加的空格数量

**Q: 支持中文吗？**
A: ASCII 艺术字主要支持英文和数字，中文需要更复杂的字符集

**Q: 如何导出为图片？**
A: 可以使用截图工具，或使用专门的终端到图片转换工具
